La prueba de Wilcoxon es una prueba estadística no paramétrica que se utiliza para comparar dos grupos relacionados o muestras pareadas. También se conoce como prueba de rangos con signo de Wilcoxon y es una alternativa a la prueba t de Student para muestras relacionadas cuando no se puede asumir normalidad en los datos. 
Tipos de pruebas de Wilcoxon:
Prueba de los rangos con signo de Wilcoxon:
Se utiliza para comparar dos muestras relacionadas, como mediciones antes y después de un tratamiento en el mismo grupo de individuos. 
Prueba de suma de rangos de Wilcoxon (o prueba de Mann-Whitney):
Se utiliza para comparar dos muestras independientes, como la diferencia entre dos grupos diferentes. 
¿Cuándo se utiliza?
La prueba de Wilcoxon es útil cuando se cumplen las siguientes condiciones: 
Los datos son ordinales o continuos, pero no siguen una distribución normal. 
Las muestras son dependientes (relacionadas o pareadas). 
No se puede asumir que las diferencias entre las muestras sigan una distribución normal. 
¿Cómo funciona?
La prueba de Wilcoxon asigna rangos a las diferencias entre las observaciones pareadas, considerando tanto la magnitud como el signo de la diferencia. Se calculan los rangos de las diferencias absolutas y luego se consideran los signos originales para obtener las sumas de los rangos positivos y negativos. El estadístico de prueba se basa en estas sumas de rangos, y se utiliza para determinar si hay diferencias significativas entre los grupos. 
En resumen: La prueba de Wilcoxon es una herramienta valiosa en estadística para comparar datos relacionados cuando no se puede asumir la normalidad, ofreciendo una alternativa robusta a la prueba t de Student.
