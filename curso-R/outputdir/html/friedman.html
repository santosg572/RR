La prueba de Friedman es una prueba no paramétrica que se utiliza para comparar tres o más grupos relacionados o apareados. Es una alternativa no paramétrica al ANOVA de medidas repetidas cuando no se cumplen los supuestos de normalidad. Se utiliza para determinar si existen diferencias significativas entre las distribuciones de las k muestras relacionadas. ¿Cuándo se usa? Cuando se tienen datos de tres o más grupos relacionados (como mediciones repetidas en el mismo sujeto). Cuando no se puede asumir que los datos siguen una distribución normal. Cuando la variable dependiente es ordinal (escala de clasificación) o continua, pero no se cumplen los supuestos de normalidad. ¿Cómo funciona? 1. Rango de los datos: Se asignan rangos a los datos dentro de cada grupo relacionado (de menor a mayor). 2. Suma de rangos: Se calculan las sumas de los rangos para cada grupo. 3. Estadístico de prueba: Se utiliza un estadístico de prueba basado en las sumas de rangos para determinar si existen diferencias significativas entre los grupos. 4. Comparaciones múltiples: Si se encuentra una diferencia significativa, se pueden realizar comparaciones múltiples por pares para identificar qué grupos son diferentes. Hipótesis: Hipótesis nula: No hay diferencia significativa entre las k muestras relacionadas. Hipótesis alternativa: Existe al menos una diferencia significativa entre dos de las k muestras relacionadas. Ventajas: No requiere la suposición de normalidad. Puede manejar datos ordinales. Es una herramienta útil para analizar datos de medidas repetidas. Software: La prueba de Friedman se puede realizar con varios programas estadísticos, incluyendo: SPSS, R, Excel (con complementos como XLSTAT), Otros programas estadísticos. Ejemplo: Un investigador quiere comparar la eficacia de tres tratamientos diferentes en el mismo grupo de pacientes. Se mide la mejora en el dolor de cada paciente después de cada tratamiento. Como los datos son de medidas repetidas (mismas personas para cada tratamiento) y puede no cumplirse la normalidad, se utiliza la prueba de Friedman para determinar si hay diferencias significativas entre los tres tratamientos.
